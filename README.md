# 红楼梦高考地图

这是一个面向高中语文整本书阅读的本地学习网站，重点服务《红楼梦》的章节理解、人物关系和高考答题训练。

## 使用方式

推荐在本目录启动一个本地服务：

```bash
python3 -m http.server 8765
```

然后浏览器打开：

```text
http://localhost:8765
```

## 内容结构

- `index.html`：网站首页。
- `assets/styles.css`：视觉样式。
- `assets/app.js`：章节筛选、详情切换、关系图等交互。
- `data/honglou_data.json`：网站使用的结构化章节、人物和专题数据。
- `data/raw_chapters/`：下载到本地的 120 回原文。
- `scripts/build_honglou_data.py`：重新下载和生成数据的脚本。

## 当前版本

第一版采用“重点章节精讲 + 全部章节可查”的方式：

- 前五回和常考重点回已经写入高考导向摘要。
- 全 120 回均可点击查看回目、原文摘录、人物命中和考点方向。
- 后续可以继续把每一回扩写成完整的“主要情节、人物关系变化、可能考法、答题提示”。

文本来源：维基文库《紅樓夢》。
